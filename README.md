# Grand Carpet AI

Endüstriyel halı desinatör otomasyon sistemi. Fotoğrafları makineye hazır dijital desenlere dönüştürür.

## Özellikler
- **Renk İndirgeme:** Fotoğraftaki renkleri hedef sayıya indirir
- **Pikselleştirme:** Tarak/atkı parametreleriyle makine ızgarasına dönüştürür

## Çalıştırma
```bash
pip install -r requirements.txt
streamlit run grand_carpet_app.py
```
